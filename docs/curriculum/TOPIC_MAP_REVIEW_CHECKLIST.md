## Overview

This checklist provides curriculum experts with a standardized framework to evaluate topic maps for alignment with official CAPS specifications and educational best practices.

**Total Maps to Review**: 50  
**Review Groups**:
- Foundation Phase: 5 scopes (Grade R, 1-3)
- Intermediate Phase: 20 scopes (Grade 4-6)
- Senior Phase: 25 scopes (Grade 7-9)

## Topic Map Structure Validation

For each topic map, verify the following structural elements:

### 1. Scope Identification

- [ ] **Scope ID**: Correctly formatted (e.g., `grade7_mathematics_en`)
- [ ] **Grade Level**: Matches assigned grade (R, 1-9)
- [ ] **Phase**: Correctly assigned (foundation, intermediate, senior)
- [ ] **Subject**: Matches CAPS subject area
- [ ] **Language**: English (or other language as appropriate)

**Pass Criteria**: All identifiers match official CAPS classification

---

### 2. Topic Hierarchy Structure

- [ ] **Root Topics**: Matches CAPS main learning areas
- [ ] **Sub-topics**: Logically grouped under root topics
- [ ] **Topic Depth**: 2-3 levels (not too shallow, not too deep)
- [ ] **Leaf Nodes**: Specific enough for lesson generation
- [ ] **No Orphan Topics**: All topics connected to hierarchy

**Example Valid Hierarchy** (Mathematics):
```
Numbers & Operations
├── Whole Numbers
│   ├── Place Value
│   ├── Addition & Subtraction
│   └── Multiplication & Division
├── Fractions & Decimals
│   ├── Understanding Fractions
│   └── Decimal Equivalents
└── Number Relationships
```

**Pass Criteria**: Hierarchy is 2-3 levels deep with logical grouping

---

### 3. Topic Content Quality

For each topic in the map, verify:

- [ ] **Clear Title**: Describes the learning objective unambiguously
- [ ] **Accurate Terminology**: Uses official CAPS vocabulary
- [ ] **Grade Appropriate**: Matches expected cognitive complexity
- [ ] **Measurable Learning Outcomes**: Can be assessed
- [ ] **Age Appropriate**: Content suitable for grade level

**Bad Examples** (Flag for revision):
- ❌ "Number stuff" (too vague)
- ❌ "Advanced calculus" (too advanced for foundation)
- ❌ "Democracy principles" (for Grade R learners)

**Good Examples** (Approve):
- ✅ "Adding 2-digit numbers with regrouping" (specific, measurable)
- ✅ "Identifying basic geometric shapes" (appropriate for Grade 1)
- ✅ "Exploring simple ecosystems" (appropriate for Grade 4)

**Pass Criteria**: All topics have clear, grade-appropriate titles

---

### 4. Topic Map Completeness

- [ ] **All Major Topics Included**: No obvious gaps in CAPS coverage
- [ ] **No Missing Concepts**: All grade-level essentials present
- [ ] **Alignment Check**: Verify against official CAPS document
- [ ] **Connection to Real-World**: Topics relate to learner experiences
- [ ] **Assessment Linkage**: Each topic aligns with assessment standards

**Completeness Audit**:
1. Read official CAPS source document for the subject
2. Identify all major learning outcomes
3. Verify each outcome has a corresponding topic in map
4. Flag any missing concepts

**Pass Criteria**: ≥95% of CAPS topics represented in map

---

### 5. Topic Progression & Sequencing

- [ ] **Logical Order**: Topics build on each other appropriately
- [ ] **Prerequisite Clarity**: Prerequisites identified and correct
- [ ] **No Circular Dependencies**: Topics don't create circular prerequisites
- [ ] **Readiness Indicators**: Clear when learner is ready for next topic
- [ ] **Spiral Curriculum**: Concepts revisited at increasing complexity

**Example Valid Progression** (Mathematics):
1. Count to 10 (Grade R)
2. Count to 20 (Grade 1)
3. Count to 100 (Grade 2)
4. Place value (Grade 3)
5. Two-digit operations (Grade 4)

**Pass Criteria**: Topics follow logical learning progression

---

### 6. Alignment with CAPS Standards

- [ ] **Learning Outcomes Alignment**: Each topic aligns with official LOs
- [ ] **Content Coverage**: Breadth and depth match CAPS specifications
- [ ] **Skills Development**: Targets development of required competencies
- [ ] **Values Integration**: Includes CAPS values (social justice, etc.)
- [ ] **Differentiation**: Acknowledges diverse learner needs

**CAPS Principles to Verify**:
- ✓ Social justice and equity
- ✓ Learner-centered pedagogy
- ✓ Practical application
- ✓ Cross-curricular connections
- ✓ Life skills integration

**Pass Criteria**: Topic map reflects CAPS philosophy and principles

---

### 7. Cross-Curricular Connections

- [ ] **Subject Integration**: Topics can connect to other subjects
- [ ] **Life Skills Links**: Connections to life skills identified
- [ ] **Values Education**: Opportunities for values integration
- [ ] **Contextual Relevance**: Topics connect to learner contexts
- [ ] **Real-World Applications**: Practical uses identified

**Example Cross-Curricular Links**:
- Mathematics + Natural Sciences: Measurement and data
- Languages + Social Sciences: Cultural literature and history
- Life Skills + All subjects: Numeracy, literacy, problem-solving

**Pass Criteria**: Minimum 2-3 cross-curricular connections identified

## Quality Standards for Approval

A topic map is **APPROVED** when:

✅ All structural elements valid  
✅ All topics clearly defined and grade-appropriate  
✅ ≥95% CAPS coverage  
✅ Logical learning progression evident  
✅ Clear alignment with CAPS standards and outcomes  
✅ Minimum 2-3 cross-curricular connections identified  
✅ No major gaps or issues  
✅ Signed off by curriculum expert  

---

## Common Issues & Solutions

### Issue: Missing topics from CAPS document
**Solution**: Add missing topics to appropriate hierarchy level, re-sequence if needed

### Issue: Topics too granular/not granular enough
**Solution**: Adjust topic hierarchy to 2-3 levels, combine or split as needed

### Issue: Topics out of sequence
**Solution**: Re-order based on learning progression, identify prerequisites

### Issue: Vague topic titles
**Solution**: Rewrite with specific learning outcomes (use verbs: identify, analyze, create)

### Issue: No assessment connection
**Solution**: Add reference to what assessment would look like for topic

### Issue: Cultural insensitivity
**Solution**: Revise topic language, add diverse examples and contexts
